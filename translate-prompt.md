# 漫畫翻譯指令

## 輸入格式

輸入為一個 JSON 檔案路徑，內容格式如下：
```json
[
  {"filename": "page01.jpg", "index": "1", "text": "日文原文"},
  ...
]
```

## 翻譯步驟

### 1. 讀取術語表
讀取 `C:/code/JPocr/glossary.txt`。
解析其中非空白、非 # 開頭的每一行，格式為 `日文原文\t繁體中文譯文`，建立術語對照表。
若術語表為空或無有效條目，繼續執行（不中斷）。

### 2. 讀取待翻譯資料
讀取輸入 JSON 檔案，取得所有待翻譯項目。

### 3. 翻譯規則
- 翻譯目標語言：**繁體中文**
- 若術語表中有對應詞彙，**必須優先使用術語表的譯文**
- 遇到以下情況，將 `ai_message` 填入說明文字（非空字串）：
  - 文字模糊、破碎，無法確定語意
  - 含有大量亂碼或無法辨識的字元
  - 語意極度曖昧，有多種可能解釋
  - OCR 明顯辨識錯誤（例如文字邏輯不通）
  - `ai_message` 說明翻譯困難的原因或可能的替代譯法，**不要**將備注混入 `translation` 欄位中
- 其餘情況 `ai_message` 設為空字串 `""`

### 4. 輸出格式

**只輸出純 JSON 陣列，不要任何說明文字、不要 markdown 包裝（不要 ```json）**：
```
[
  {"filename": "page01.jpg", "index": "1", "translation": "繁體中文翻譯", "ai_message": ""},
  {"filename": "page01.jpg", "index": "2", "translation": "暫譯", "ai_message": "原文疑似OCR錯誤，語意不明，提供暫譯供參考"},
  ...
]
```
